Project Description
SheFix is a women-focused safety and skill-building app that empowers women to handle everyday challenges independently. From step-by-step DIY guides like changing a tire or fixing a tap, to emergency safety tools and community support, SheFix helps women feel confident, prepared, and protected in any situation.

The Problem Statement
Many women — especially college students living independently — face everyday problems like:
- Flat tyres
- Blown fuses or electrical issues
- Health concerns
- Safety worries
- Lack of quick guidance
Often, there is:
- Dependency on others
- Hesitation to ask for help
- No trusted source for quick solutions

SheFix provides this instant, empowering support.

The Solution
SheFix solves these issues by offering:
- Easy, safe, step-by-step tutorials for technical tasks
- Health & wellness guidance
- Beauty & self-care tips
- Emergency support features
- AI chatbot for everyday questions

Women can interact with the platform to get help for big problems to small ones confidently and safely.
